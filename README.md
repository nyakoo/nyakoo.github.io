# N2日语助手
一款纯前端网页工具，用于记忆日语单词和文法知识点，基于 GitHub Pages 免费部署。

## 在线访问
https://nyakoo.github.io/

## 项目简介
- 同时集成单词和文法两个练习模块；
- 支持自定义 txt 单词词库和文法知识点；
- 全静态 HTML+JS 网页，数据仅保存在本地。

## 核心功能
### 1. 双种文本加载方式
-  **虚拟数字九宫格加载（在线Pages专用）**
   在仓库 `WordTxt` 和`BunpoTxt`文件夹存放 `1.txt`、`2.txt` 数字命名词库，页面输入数字一键读取对应文本；
-  **本地文件上传（离线本地打开可用）**
   直接上传电脑内 txt 文件，无需部署也能使用。

### 2. 单词练习模块
- **假名挑战**：给出日语原词，输入假名/罗马音作答；
- **书写测试**：显示中文释义，依次点击乱序字块拼出日语原词；
- **连线消除**：中日单词配对连连看，轻松巩固词汇。

### 3. 文法专项攻克
- **例句填空**：给出挖空句子，根据提示词和语境，依次点击下方乱序字块拼出完整句子。
- **接续搭配**：提供语法点和当前考察的词性，学习者在下方若干选项中选取其正确的接续形态。
- **释义配对**：卡片实时读取日语语法核心，学习者在四个选项中点击正确的中文释义作答。

### 4. 复习优化机制
- 答错/寻求过帮助的知识点将自动收录到「难点列表」；
- 通关后可选择：重刷本关 / 只刷难点；
