# N2日语单词助手
一款纯前端离线日语单词背诵网页工具，基于 GitHub Pages 免费部署。

## 在线访问
https://nyakoo.github.io/tango.html

## 项目简介
- 本工具专为N2日语学习者打造；
- 支持自定义 txt 单词词库；
- 内置「学习模式」、「假名挑战」、「书写测试」、「连线消除」共4个练习模块；
- 全静态 HTML+JS 网页，数据仅保存在本地。

## 核心功能
### 1. 双种词库加载方式
-  **虚拟数字九宫格加载（在线Pages专用）**
   在仓库 `WordTxt` 文件夹存放 `1.txt`、`2.txt` 数字命名词库，页面输入数字一键读取对应单词本；
-  **本地文件上传（离线本地打开可用）**
   直接上传电脑内 txt 文件，无需部署也能使用。

### 2. 四大练习模块
- **学习模式**：完整展示日语原词、假名/罗马音、声调、释义，支持边学习边输入假名和自动切卡；
- **假名挑战**：给出日语原词，输入假名/罗马音作答；
- **书写测试**：给出中文释义，依次点击乱序字块拼出日语原词；
- **连线消除**：中日单词配对连连看，轻松巩固词汇。

### 3. 复习优化机制
- 答错/寻求过帮助的单词将自动收录到「难词列表」；
- 通关后可选择：重刷本关 / 只刷难词；
- 罗马音自动转假名输入，支持标准五十音罗马音拼写规则。

## 词库 txt 格式规范
- 每行1个单词，以半角逗号分隔，依次为单词、读音、声调、含义；
- 编码必须为 **UTF-8**；
- 举例：毛糸,けいと,0,毛線
